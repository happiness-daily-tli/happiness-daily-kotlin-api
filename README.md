# Code & Commit Convention
- 코틀린스럽게 작성 하는것을 지향한다.
- PR템플릿을 적절히 활용하여 작성하자.
- 최소한의 기능당 테스트코드 1개를 작성하여 PR을 날린다.
 
```

- domain
- common
```
